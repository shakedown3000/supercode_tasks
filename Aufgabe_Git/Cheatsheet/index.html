Dies ist ein Cheatcheat für Git

Befehl Beschreibung
git init Initialize an existing directory as a Git repository
git add Add a file as it looks now to your next commit (stage)
git commit -m
“[descriptive message]”		 Commit your staged content as a new commit snapshot
git push -u origin main Local changes are pushed to the online repository.
git pull Fetch and merge any commits from the tracking remote branch
